# Windsurf Demo App

A demo application and tasks to demonstrate the power of the [Windsurf Editor](https://windsurf.ai/).

The application comes pre-loaded with a variety of workflows that will give you hands-on experience with Windsurf basics.

Topics covered include: 
- Intro to Cascade
- Model Selection
- Managing Conversations
- Working with Cascade
- Adding Context
- Terminal Usage
- Rules
- Workflows
- Use Case: Bug Fixing
- Use Case: Feature Development

To get started, simply clone the repo, open it in Windsurf, and type '/intro' into the Cascade input! 

## Prerequisites

- Python 3.7 or higher
- Node.js and npm

## Getting Started

1. Clone the repository:
```bash
git clone <repository-url>
cd windsurf-demo
```

2. Install Python dependencies:
```bash
pip3 install -r requirements.txt
```

3. Start the Flask server:
```bash
python3 app.py
```

4. Open your web browser and navigate to:
```
http://localhost:5000
```

5. Play the game!
